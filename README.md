# 7 days of js with scrimba.com
## Day 1 - Add border

https://scrimba.com/c/cPkE44uE?utm_source=newsletter&utm_medium=email&utm_campaign=jschallenge_email

### Goal

Given rectangular matrix of characters,add a border of asterisks "*" to it.

For example:

```
array = ['abc', 'xyz']
```

result:

```
arrayWithBoder = [
  '*****',
  '*abc*',
  '*xyz*',
  '*****'
]
```
